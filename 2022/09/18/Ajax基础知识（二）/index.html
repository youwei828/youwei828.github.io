Ajax基本知识（二）

2022/9/18 Ajax基本知识 JavaScript GET POST

一、使用 Ajax完成前后端通信

完整栗子：

const url = 'https://www.imooc.com/api/http/search/suggest?words=js';
const xhr = new XMLHttpRequest();
xhr.onreadystatechange = () => {
    if (xhr.readyState < 4) return;
    if ((xhr.status >= 200) & (xhr.status < 300) || xhr.status === 304) {
        console.log(xhr.response);
        console.log(typeof xhr.response);
        }
    };
xhr.open('GET', url, true);
xhr.send(null);

二、GET

1、携带数据

GET请求不能通过请求体来携带数据，但可以通过请求头携带。

这种请求方式的提交在一定程度上和表单提交很相似。

2、数据编码

如果携带的数据是非英文字母的话，比如说汉字，就需要编码之后再发送给后端，不然会造成乱码的问题，可以使用encodeURlComponent()编码（POST请求同理）。

栗子（以模板字面量的格式）：

const url =`https://www.imooc.com/api/http/search/suggest?words=${encodeURLComponent('前端')}`

**NOTE:**因为不同的浏览器有不同的编码格式，服务器只有一套固定编码的方式，所以解码出来就不一定是一样的内容，我们在开发的时候要自己先进行编码。

三、POST

1、携带数据

POST请求主要通过请求体携带数据，同是也可以通过请求头携带。

const url = 'https://www.imooc.com/api/http/search/suggest?words=js';
const xhr = new XMLHttpRequest();
xhr.onreadystatechange = () => {
    if (xhr.readyState < 4) return;
    if ((xhr.status >= 200) & (xhr.status < 300) || xhr.status === 304) {
        console.log(xhr.response);
        console.log(typeof xhr.response);
        }
    };
xhr.open('POST', url, true);
//如果想要发送数据，直接写在send()的参数位置，一般是字符串。
xhr.send('username=zs&age=18');
//如果要传对象的话，需要先把对象转化成字符串的形式，这里用到了JSON的格式。
xhr.send();
